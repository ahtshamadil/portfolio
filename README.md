# Portfolio — ahtshamadil.me

Personal portfolio site built with **Next.js 16**, **TypeScript**, and **Tailwind CSS v4**. Features an interactive 3D scene via React Three Fiber, smooth Framer Motion transitions, and a working contact form powered by EmailJS.

**Live:** https://ahtshamadil.me

---

## Tech Stack

- **Framework:** Next.js 16 (App Router) + React 19
- **Language:** TypeScript
- **Styling:** Tailwind CSS v4
- **3D:** React Three Fiber + Drei + Three.js
- **Animation:** Framer Motion
- **Forms:** react-hook-form + EmailJS
- **Icons:** react-icons

---

## Features

- Interactive 3D hero scene
- Smooth page transitions and scroll animations
- Responsive, mobile-first layout
- Working contact form (no backend required — EmailJS)
- Strongly typed components and props

---

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Available Scripts

| Command | What it does |
| --- | --- |
| `npm run dev` | Start the development server |
| `npm run build` | Production build |
| `npm start` | Run the production build |
| `npm run lint` | Run ESLint |
